FontAwesome

Hubble ships with a complete glyph-icon library thanks to FontAwesome. The library is free and comes with 675 individual icons.

To create an icon, use a <span> element with the class .glyph-icon in addition to .glyph-icon-* and the icon name as per the FontAwesome documentation.

Icons can be sized using the .icon-xs, .icon-md, .icon-lg, .icon-xl classes.

For a full list of available icons, please see the FontAwesome documentation.
