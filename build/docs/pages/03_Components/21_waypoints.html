Waypoints

Waypoints provide a convienient to automatically scroll to content on a page. They also provide the ability to land at a particlar element on page load via the URL hash.

  • Usage
  • Options

Usage

To setup a click-to-scroll event, simply add the .js-waypoint-trigger class to a clickable element, like a button

Any element with a data-waypoint attribute provides the ability to land at that element on page load using the URL hash. For example, http://example.com#landhere on pageload would scrol to the element with data-waypoint="landhere".

Options

You can specify the the animation speed as well as the animation easing via HTML data- arrtibutes on the click-target. data-waypoint-speed sets the scroll animation in miliseconds, while data-waypoint-easing sets the scroll animation easing via a valid CSS easing property.
