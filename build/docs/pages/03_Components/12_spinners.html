Spinners

Loading spinners come in handy for everything from a page-loading animation to Ajax requests.

  • Basic example
  • Styles

Basic example

Loading spinners use SVG and some simple CSS animations. As such, the markup is very minimalist.

Use .hidden class to hide any element when it doesn't need to be visible.
